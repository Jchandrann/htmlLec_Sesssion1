Steps to make a folder a REPO

Install git
git clone 'Link of url created a repo in GITHUB' - to clone the remote repo
changed the branch from main to gh-pages created it GITHUB git checkout gh-pages

git status - to view current of repo

git add . - staging repo for commit

git commit -m 'commit msg' - commiting the changes to repo

git push - to push repo from local to online github repo

Configuring new github account in local folder used PAT "personnal acess token" In credential manager of window created a generic credential. git:https://github.com given user name and password is PAT generated from GIThub->settings->PAT--> fill form --> generate token. source:
